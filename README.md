# Pump Modeling
Modeling of a centrifugal water pump, for a steam reforming plant, using [OpenFoam v2206](https://www.openfoam.com/news/main-news/openfoam-v2206):

- 7Mln cells mesh,
- MRF zone, 
- $\Delta p = 2.1 bar$,
- $\omega = 160 rad/s$,
- $\eta = 0.6$,
- $Q = 1.097*10^{-2} m^3/s$
- $D_{rotor} = 22.3cm$
